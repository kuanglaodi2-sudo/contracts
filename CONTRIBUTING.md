# Contributing to Contracts

Welcome! This guide helps you contribute to the Contracts project.

## Getting Started

1. **Fork** this repository
2. **Clone** your fork: `git clone https://github.com/YOUR_USERNAME/contracts.git`
3. **Create** a branch: `git checkout -b your-feature-name`

## Development Setup

```bash
# Install dependencies
npm install

# Run tests
npm test
```

## Code Style

- Use meaningful variable names
- Add comments for complex logic
- Follow existing code patterns

## Submitting Changes

1. Commit your changes: `git commit -m "Add feature description"`
2. Push to your fork: `git push origin your-branch-name`
3. Create a **Pull Request** to the main repository

## PR Guidelines

- Include a clear description of changes
- Reference any related issues
- Ensure tests pass before submitting

## Rewards

This project has an active bounty program. Check the [bounty board](https://github.com/Scottcjn/rustchain-bounties) for tasks with RTC rewards.

## Questions?

- Open an issue for bugs or feature requests
- Join the community discussions
